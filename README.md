# Professional ATM Banking Simulation

A clean and efficient terminal-based banking system built with Python. This project simulates core banking operations with a focus on security and user experience.

---

##  Features

* **Secure Authentication**: Includes a PIN-based login system that blocks the card after 3 failed attempts.
* **Balance Inquiry**: Real-time display of current account funds.
* **Deposit System**: Allows users to add positive amounts to their balance with input validation.
* **Smart Withdrawal**: Features a menu for quick-cash (£10, £20, £50) or custom amounts in multiples of £10.
* **Error Handling**: Prevents invalid inputs, negative deposits, and overdrafts.
